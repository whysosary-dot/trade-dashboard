설정 / API 키

실데이터 연결을 위한 API 키를 등록하세요
🇰🇷 관세청 수출입무역통계 (공공데이터포털)
무료 · 가입 후 10~20분 내 발급 · 국내 HS코드 수출입 데이터
활용신청 페이지
  1. 공공데이터포털 가입
    data.go.kr 회원가입 (공동인증서 · 간편로그인 가능)
  2. "관세청_수출입무역통계" 검색
    상단 링크 클릭 또는 data.go.kr 에서 "수출입무역통계" 검색
  3. 활용신청 클릭
    용도에 "개인 참조" 선택, 트래픽 1,000회/일 이상 신청 추천
  4. 일반 인증키(Decoding) 복사
    마이페이지 → 개발계정 → 일반 인증키 확인
  5. 아래 입력란에 붙여넣기 → 저장
샘플 엔드포인트
apis.data.go.kr/1220000/Itemtrade/getItemtradeList?hsSgn=854232&strtYymm=202501&endYymm=202612&serviceKey=YOUR_KEY
🇺🇸 U.S. Census Bureau
무료 · 이메일 인증 즉시 발급
  1. 우측 "키 발급" 버튼 클릭
  2. 이름 · 조직명(개인) · 이메일 입력
  3. 이메일 수신 → 링크 클릭으로 인증
  4. 이메일에 담긴 40자리 키 복사 후 저장
키 발급
🇪🇺 Eurostat
무료 · 키 불필요
Eurostat API는 별도 가입이나 키가 필요 없습니다. 공개 REST 엔드포인트를 바로 호출합니다.
ec.europa.eu/eurostat/api/dissemination/statistics/1.0/data/DS-059668?...
현재 등록 상태
💡 키를 저장한 뒤 [미국 HS] 또는 [한국 HS] 탭에서 HS 코드를 입력하면 해당 코드의 실데이터를 브라우저가 직접 호출합니다 (별도 서버 불필요).
🔒 키는 브라우저 localStorage 에만 저장되며 외부로 전송되지 않습니다.
🇰🇷 관세청은 CORS 정책으로 공용 프록시 corsproxy.io 를 경유합니다.
사용 팁
· 후행 지표입니다. 미국·유럽 데이터는 보통 1~2개월 지연되어 공개됩니다.
· HS코드는 6자리까지 국제 공통, 그 뒤는 국가별로 다릅니다. (미국: 10자리, 한국: 10자리)
· 반도체처럼 미국이 전략산업으로 지정한 품목은 M3에서 비공개이지만, 수출입 데이터로는 확인 가능합니다.
· 유럽(Eurostat)·NAICS·M3 는 현재 샘플 데이터로 표시됩니다 (업스트림 엔드포인트 변경 중).
